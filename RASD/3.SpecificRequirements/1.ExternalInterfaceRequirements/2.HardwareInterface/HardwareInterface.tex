\subsection{Hardware Interfaces\label{sect:3.1.2}}
	\begin{itemize}
\item Web applications must be accessible by \ic{Municipality} using a computer with characteristics specified in \hyperref[sect:3.4.2]{Section 3.4.2}. 
\item Smartphone on which the app will work must provide to the app an Internet connection used to send \ic{User reports} to SafeStreets servers and must have a GPS antenna built in. 

\end{itemize}
